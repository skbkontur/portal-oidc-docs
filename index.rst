Что такое Контур.ID
------------------------

Контур.ID — это система единой аутентификации, которая работает на базе протоколов OpenID Connect и OAuth2.0.   

Основные понятия, которые важны для интеграции:

   *Что такое аутентификация? Это когда вам надо узнать, какой именно пользователь к вам пришел, и проверить его подлинность*.

   *Что такое авторизация? Это процесс проверки наличия прав доступа на выполнение какой-либо операции*.  

Зачем использовать Контур.ID
---------------------------------

Интеграция с Контур.ID позволит вам автоматизировать работу вашей информационной системы:

* Для определения пользователя, который пришел в ваше приложение, достаточно настроить его взаимодействие с OpenID Провайдером. Для входа в ваше приложение пользователь предъявляет свою учетную запись Контура или регистрирует новую. Вся аутентификация будет происходить в браузере.

* При помощи OpenID Провайдера также можно получить разрешение у пользователя на доступ к его данным из продуктов Контура. 

* Для web-сервисов и desktop-приложений доступны способы входа по логину/паролю, по сертификату, по телефону, по почте. Для мобильных приложений можно настроить вход только по логину/паролю, по телефону или по почте.
  
* Автоматически взаимодействовать с API Контура без участия пользователя.

Интеграционные возможности
--------------------------

**Инфраструктура для конечных пользователей**

* Так как у Контура большая база клиентов и много уже зарегистрированных пользователей, вероятно ваши пользователи уже являются клиентами Контура. Им не придется регистрировать новую учетную запись. 

* У каждого пользователя с появлением учетной записи открывается личный кабинет, где он может настраивать свои данные и управлять доступами к данным.

* Также в личном кабинете пользователи смогут настроить для себя способ входа: по телефону, по логину, а также они могут использовать вход по сертификату.

* Помимо способа входа в личном кабинете пользователь может включить двухфакторную аутентификацию. И второй фактор также настраивается пользователем. Это может быть код в sms или подтверждение в специальном приложении, которое работает даже без доступа к Интернету. 

**Инфраструктура для интегратора**

* Интеграция с Контур.ID дает вам доступ к экосистеме продуктов Контура, а также возможность реализации единого входа (Single Sign-On).

* Вы сможете взаимодействовать с публичными API продуктов Контура автоматически или от имени пользователей. 

* При интеграции с Контур.ID или любым продуктом Контура вам будет открыт `личный кабинет интегратора<https://integrations.kontur.ru/>`.

С чего начать
-------------

Перед интеграцией с Контур.ID по протоколу OpenID Connect ознакомьтесь с документацией и выберите:

   1. Сценарий интеграции, который подходит вам. Сценарий даст ответ на вопрос "зачем нужна интеграция с Контур.ID или что хотите реализовать?"
   2. Схему интеграции. Схема даст ответ на вопрос "как реализовать интеграцию?" Схема взаимодействия с провайдером зависит от типа вашего приложения: с серверной частью или без. Выбирайте схему по вашему сценарию и типу приложения.

Зарегистрируйте ваше приложение. При регистрации нужны данные, которые будут использованы в реализации схемы интеграции.

Реализуйте получение токенов по выбранной схеме интеграции.
